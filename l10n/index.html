How to test translations

(1) Web App Testing:

Translations for the web app can be tested in a special testing environment that gets refreshed 1-2 hours after translations have been added in Crowdin. Press the button below to go to the web app testing environment. Note: this environment is clean so you will need to create a new account in order to test.

Test Web App → l10n.aliasvault.com

(2) Browser Extension & (3) Mobile Apps:

For the browser extension and mobile apps, special nightly builds are generated automatically several times per day that contain the latest translations. These builds are available for download below and can be installed manually. It can take up to 6 hours for builds to be refreshed after making changes in Crowdin.

Available test builds

Browse and download the latest versions with their build dates (UTC). For information on how to manually install the browser extension and mobile apps, see the technical documentation.

⚠️ Important warning: Some beta app versions may contain data model updates. We strongly recommend using a test account instead of your real account. You can use a test account either on the l10n.aliasvault.com/api test API server or on the aliasvault.com production server. Do not use your real account, as updating your vault with these beta builds may prevent you from logging in with the current live apps.
Note: When manually installing the Android app APK, you will need to uninstall the existing app (installed via the Google Play Store) first.
