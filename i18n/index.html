We appreciate your help in making AliasVault accessible to people all over the world. Your contributions to translating the app into more languages help us reach and support a global community.

How to Test Translations

(1) Web App Testing:

Translations for the web app can be tested in a special testing environment that gets refreshed 1-2 hours after changes have been made in Crowdin. Press the button below to go to the web app testing environment. Note: this environment is clean so you will need to create a new account in order to test.

(2) Browser Extension & (3) Mobile Apps:

For the browser extension and mobile apps, special nightly builds are generated automatically several times per day that contain the latest translations. These builds are available for download below and can be installed manually.

Note: Translations made on Crowdin can take up to 12 hours to appear in test builds, as apps are typically built a few times per day.
