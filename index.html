AliasVault Downloads

Official Downloads Server

Welcome to the official AliasVault downloads server. Here you can find the latest releases and other official binaries for AliasVault products.

1. Official Releases

Find all app releases here. This directory contains the stable, production-ready versions of AliasVault that are suitable for general use. These releases mirror the official releases from the GitHub repository.

2. Nightly Builds

These are periodic automatic builds of the main branch. Nightly builds are generated automatically on a regular schedule (multiple times per day) and include the latest code changes from the main development branch. These builds are intended for developers and advanced users who want to test new features or bug fixes before they are included in official releases. Please note that nightly builds may be unstable and are not recommended for production use.

3. Crowdin Translation Testing

These builds contain the latest translations made in Crowdin and allow for testing translation changes before they are merged into the main branch. These builds are specifically designed for translators, localization teams, and users who want to verify that translations appear correctly in the application interface. They include all pending translation updates from the Crowdin project, enabling early detection of issues such as text overflow, missing translations, or formatting problems. Help us improve AliasVault by testing these builds in your language!
